goConcurrent()

goConcurrent(array  $requests) : array

并行POST或者Get请求协程（自动完成DNS查询、获取数据）

Parameters

array $requests

格式如： 

[
  [
    // 必须为全路径URL
    'url'         => 'http://www.baidu.com/xxx',
    'method'      => 'GET',
    'dns_timeout' => 1000, // 默认为30s
    'timeout'     => 3000,  // 默认不超时
    'headers'     => [],    // 默认为空
    'data'        => ['a' => 'b'] // 发送数据
    ],
  [
    'url'         => 'http://www.baidu.com/xxx',
    'method'      => 'POST',
    'timeout'     => 3000,
    'headers'     => [],
    'data'        => ['a' => 'b'] // 发送数据
    ],
  [
    'url'         => 'http://www.baidu.com/xxx',
    'method'      => 'POST',
    'timeout'     => 3000,
    'headers'     => [],
    'data'        => ['a' => 'b'] // 发送数据
    ],
];
